# 🐍 Estudos de Python

Este repositório foi criado para armazenar meus estudos, exercícios, projetos e anotações sobre **Python**.

O objetivo é centralizar todo o conteúdo que estou aprendendo, acompanhar minha evolução e servir como material de consulta para projetos futuros.

---

## 🎯 Objetivos

- Aprender os fundamentos da linguagem Python.
- Desenvolver a lógica de programação.
- Praticar resolução de problemas e algoritmos.
- Conhecer as principais estruturas de dados.
- Aprender boas práticas de programação.
- Criar uma base sólida para desenvolvimento de projetos.
